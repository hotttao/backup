Vue 项目实战，实现一个 Xmall 的购物商城
<!-- more -->

## 1. 项目构建

```bash
# 1. 项目创建
# 选择 bable，vuex vue router
vue create xmall 

# 2. 安装 element-ui
vue add element # 选择局部安装

# 3. 安装 sass
# https://cli.vuejs.org/zh/guide/css.html
cnpm install -D sass-loader sass

# 4. 安装 axios
cnpm i axios -S

# 5. 安装图片懒加载库
cnpm i vue-lazyload -S

# 6. 删除默认内置的路由和组件
```
